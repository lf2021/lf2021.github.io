注意：这里只是针对 VScode 用户

  • 安装Markdown All in One这个插件 (看下载量就知道好不好了😊)

  • 对于你要生成目录的md文件，使用VScode快捷键 ctrl + shift + p 打开界面如下：

  • 然后输入 Markdown All in One: Create Table of Contents 回车即可生成目录，是不是很方便呢

  • 上传至GitHub上也亲测有效
